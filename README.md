# 🌍 Tour Project — Інформаційна система для туристичної агенції

Даний проект є результатом виконання курсової роботи, метою якої було створення сучасної веб-платформи для пошуку, перегляду та бронювання туристичних турів. Система побудована на базі актуального технологічного стеку, що поєднує гнучкість мови Python на сервері та швидкість бібліотеки React на клієнтській стороні.

Проект вирішує завдання автоматизації роботи турагента та спрощує процес взаємодії клієнта з базою пропозицій. Користувачі мають змогу переглядати доступні напрямки, ознайомлюватися з деталями подорожей та залишати заявки на бронювання у зручному інтерфейсі.

---

## 🏗 Структура проекту

Проект розділений на два основні модулі, що дозволяє незалежно розробляти та масштабувати фронтенд і бекенд:

* **Flask_project_cursova/** — **Backend API**. Відповідає за логіку роботи з базою даних, обробку запитів користувачів та маршрутизацію даних. Побудований на фреймворку Flask.
* **React_project_cursova/** — **Frontend Client**. Клієнтський інтерфейс, реалізований як Single Page Application (SPA). Забезпечує інтерактивність та високу швидкість взаємодії.
* **venv/** — Віртуальне середовище (локально), яке містить усі необхідні залежності Python. (Ігнорується Git для чистоти репозиторію).

## 🚀 Інструкція із запуску проекту локально

Для успішного запуску переконайтеся, що на вашому комп'ютері встановлено **Python 3.x** та **Node.js**.


### 1. Запуск бекенду (Flask)

Перейдіть у папку:
```bash
cd Tour_project
```

Перейдіть у папку сервера:
```bash
cd Flask_project_cursova
```
Активуйте віртуальне середовище:
```bash
..\venv\Scripts\activate
```
Перейдіть у папку:
```bash
cd likiatours
```
Запустіть проект:
```bash
python app.py
```

### 2. Запуск фронтенду (React)

Перейдіть у папку:
```bash
cd Tour_project
```

Перейдіть у папку клієнта:
```bash
cd React_project_cursova
```

Перейдіть у папку:
```bash
cd my-first-app
```

Запустіть додаток:
```bash
npm install
npm start
```
